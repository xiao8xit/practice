# Инструкция по созданию скриншотов для сдачи работы

## 1. Скриншот Swagger документации
- Запустите сервер: `uvicorn app.main_api:app --reload`
- Откройте в браузере: http://127.0.0.1:8000/docs
- Сделайте скриншот всей страницы с видимыми эндпоинтами
- Сохраните как: `docs_screenshot.png`

## 2. Скриншот успешного запроса через Postman
- Установите Postman
- Создайте новый запрос GET на http://127.0.0.1:8000/books/
- Отправьте запрос
- Убедитесь, что код ответа 200
- Сделайте скриншот с результатом
- Сохраните как: `postman_request.png`
